# Guía de usuario de Tandas App

Bienvenido a Tandas App. Esta aplicación te ayuda a organizar tandas, registrar pagos y mantener un mejor control de tus participantes desde tu teléfono.

## 1. Qué puedes hacer con la app

- Crear y administrar tandas
- Agregar participantes
- Registrar pagos y llevar seguimiento
- Recibir recordatorios locales en tu dispositivo
- Abrir mensajes por WhatsApp de forma manual y segura
- Exportar e importar respaldos en formato JSON

## 2. Primeros pasos

1. Instala la app desde la página de descarga.
2. Abre Tandas App.
3. Ve a Configuración para activar notificaciones si deseas recordatorios.
4. Crea tu primera tanda con nombre, monto, fecha y participantes.

## 3. Crear una tanda

Para registrar una nueva tanda:

1. Toca el botón para agregar una tanda.
2. Escribe el nombre de la tanda.
3. Define el monto y la frecuencia.
4. Guarda los cambios.

Después podrás entrar al detalle de la tanda para revisar participantes, pagos y avances.

## 4. Participantes y pagos

Dentro de cada tanda puedes:

- Agregar participantes con su nombre y teléfono
- Marcar pagos como realizados
- Revisar pagos pendientes y vencidos
- Consultar el estado general de la tanda

## 5. Recordatorios y notificaciones

En la sección de Configuración puedes:

- Activar o desactivar notificaciones locales
- Configurar horarios permitidos para recordatorios
- Ajustar preferencias generales de uso

> Importante: los recordatorios se generan en este dispositivo. Si cambias de equipo o desinstalas la app, conviene exportar un respaldo.

## 6. WhatsApp

Tandas App no envía mensajes automáticos en segundo plano. Para mayor seguridad y confiabilidad, la app abre WhatsApp o WhatsApp Web para que tú revises y envíes el mensaje manualmente.

## 7. Respaldo de información

Puedes proteger tus datos desde Configuración:

- Exportar datos para crear un respaldo manual
- Importar un respaldo existente
- Activar el respaldo automático semanal al abrir la app

El formato oficial de respaldo es JSON.

## 8. Ayuda y soporte

Si necesitas ayuda:

- Usa el botón Enviar correo dentro de la app
- Consulta esta guía cuando tengas dudas de uso
- Mantén la aplicación actualizada para recibir mejoras y correcciones

## 9. Recomendaciones de uso

- Verifica los datos antes de guardar cambios
- Haz respaldos periódicos si manejas varias tandas
- Protege tu dispositivo con bloqueo o biometría
- No compartas archivos de respaldo por medios inseguros
